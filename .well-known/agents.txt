# Idempotency-Key header REQUIRED on all mutating endpoints.
# Errors are structured with stable machine-recoverable codes (16-code enum).
# MCP and REST are co-equal at every endpoint — neither transport is the
# "real" API; both pass through the same boundary publish gate.
# Per-tenant isolation is enforced by the code path itself, not by
# configuration or policy — cross-tenant data access is architecturally
# prevented at the boundary publish layer.
